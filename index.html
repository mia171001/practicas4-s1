EXPLICANDO CONCEPTOS DE LA CLASE

Para cambiar el tipo de letra de nuestro texto usamos "font-family", tenemos varias opciones en este caso usaremos "Times New Roman",si no lo encuentra usaremos "Timessi" y si no encuentra este ultimo usaremos "serif"

Para modificar el tamaño de la letra usaremos font-size, ahora explicaremos cada unidad de medida Pixeles (px): Unidad fija y absoluta. El tamaño del texto en píxeles no cambia. Por ejemplo, font-size: 16px; establece el tamaño de la fuente en 16 píxeles. Em(em): Unidad relativa al tamaño de fuente del elemento padre. Por ejemplo, si el tamaño de fuente del elemento padre es 16px, entonces 1em será igual a 16px. Rem (rem): Unidad relativa al tamaño de fuente del elemento raíz (html). Por defecto, el tamaño de fuente del elemento raíz es 16px en la mayoría de los navegadores. Por lo tanto, 1rem será igual a 16px. Al titulo le hemos aplicado 40px

"line-height" es una propiedad de CSS que controla la altura entre líneas de texto. Puedes usar valores numéricos o unidades de medida para ajustar esta altura. Por ejemplo, line-height: 3; establece la altura de línea como 3 veces el tamaño de fuente, lo que mejora la legibilidad y apariencia del texto.Lo hemos aplicado entre los parrafos

Para modificar la direccion del texto usamos "text-align" aqui podemos centrar, mover a la izquierda y derechao que se justifique. Un ejemplo es el titulo el cual esta centrado

font-weight es una propiedad de CSS que controla el grosor o la "negrita" del texto. Puedes usar valores numéricos como 400 para normal y 700 para negrita, o palabras clave como normal y bold. Por ejemplo, font-weight: bold; hará que el texto se vea más grueso y font-weight: normal; lo devolverá a su grosor predeterminado. La hemos aplicado al titulo.
